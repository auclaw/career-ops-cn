# About / 关于本项目

## 这是什么

career-ops-cn 是一个 **基于 Claude Code 的 AI 驱动求职自动化工作流**，专为中国求职市场本地化定制。

它把 Claude Code 变成你的全职求职指挥中心——不用手动维护 Excel 追踪表格，AI 帮你完成从「发现职位」到「评估匹配度」再到「定制简历」的全流程。

## 核心设计理念

> **这不是海投工具，这是你的「职位过滤器」**

- ❌ **不做批量海投**：我们不帮你 spam 几百家公司
- ✅ **只做精准筛选**：从几百个职位中帮你找出真正值得投递的少数几个
- 🛡️ **风险过滤**：新增公司背景调查维度，帮你识别诈骗、传销、招聘收费、培训贷
- 🤝 **人机协作**：AI 做信息收集和初步评估，最终是否投递由你决定

## 为什么做这个项目

求职过程中重复劳动太多了：

1. 每天刷招聘网站看到很多职位，一个个打开看很花时间
2. 每个职位都需要重新定制简历，手动改来改去效率很低
3. 有些公司听起来不错，实际是诈骗公司，需要自己查背景
4. 投完之后追踪申请状态，维护表格很麻烦

career-ops 把这些重复工作自动化了，让你把精力花在准备面试上，而不是机械劳动。

本项目基于原作者 [santifer/career-ops](https://github.com/santifer/career-ops) 进行了中国本地化改造，感谢原作者的优秀工作。

## 谁适合用这个项目

- 正在找工作的互联网/AI/游戏行业从业者
- 已经安装并会使用 Claude Code
- 想要提高求职效率，减少重复劳动
- 认同「精准投递＞批量海投」的理念
